# Geekay Intelligence Platform

## C3 (Contract Control Center)

C3 is the first application built on the Geekay Intelligence Platform.

The platform provides a reusable foundation for internal operational applications running on Microsoft 365 and SharePoint Online.

Current Status:

- ✅ Platform Foundation Complete
- ✅ Platform SDK v1.0 Frozen
